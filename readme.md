#George Plus
============
George (Formal Logic parser) enhancer.

Adds the following new features:
- Syntax Highlighting
- Code Folding
- In-Browser storage
- Proof Auto-renumbering (type ")" on a new line)
- Auto-complete
- Code snippets (type "a{, d{, and s{ are the snippets")
